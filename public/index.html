Enter amounts without thousands separators (e.g. 10.50 for 10.50 €).
How to use this form?

Double-entry rules require that total Debits (DR) must exactly equal total Credits (CR). The 'Post Transaction' button becomes active only when the entries are balanced (Difference = 0.0000).

  • 🔴 Debit (DR): Money leaves this wallet (decrease)
  • 🟢 Credit (CR): Money enters this wallet (increase)
